2. **Choose a tool** → Browse or search for what you need
3. **Upload file** → Drag & drop or click to select

## 📦 All Features

### **Organize PDF**
Manage and restructure your PDF documents with ease.

| Tool | Description | Input | Output |
|------|-------------|-------|--------|
| **Merge PDF** | Combine multiple PDFs in any order | Multiple `.pdf` | Single PDF |
| **Split PDF** | Extract specific pages into separate files | `.pdf` | Multiple PDFs |
| **Rotate PDF** | Rotate pages 90°, 180°, or 270° | `.pdf` | Rotated PDF |
| **Organize PDF** | Reorder, delete, or add pages | `.pdf` | Organized PDF |
| **Crop PDF** | Remove margins or crop specific areas | `.pdf` | Cropped PDF |

### **Optimize PDF**
Reduce file size and improve document quality.

| Tool | Description | Input | Output |
|------|-------------|-------|--------|
| **Compress PDF** | Reduce file size while maintaining quality | `.pdf` | Smaller PDF |

| Tool | Description | Input | Output |

| **PDF to Word** | Convert to editable DOCX documents | `.pdf` | `.docx` |
| **PDF to PowerPoint** | Convert to editable PPTX presentations | `.pdf` | `.pptx` |
| **PDF to Excel** | Extract data into spreadsheets | `.pdf` | `.xlsx` |

**Office → PDF:**

| Tool | Description | Input | Output |
