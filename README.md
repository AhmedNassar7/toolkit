# Toolkit

Toolkit is a simple file utility app for common PDF and document tasks.

## Features

- Merge PDF files
- Split PDF files
- Rotate PDF pages
- Compress PDF files
- Add page numbers to PDFs
- Protect PDFs
- Add watermarks to PDFs
- Convert DOCX to PDF
- Convert JPG or PNG images to PDF
- Convert PDF pages to images
- Convert PDF to Office formats

## How to Use

1. Open the app.
2. Choose a tool.
3. Upload your file.
4. Download the result.
