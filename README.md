## 📖 How to Use

### Basic Usage (All Tools)

```
1. Go to Home → Browse Tools by Category
   - Organize PDF
   - Optimize PDF
   - Convert PDF
   - Edit PDF
   - PDF Security
   - PDF Intelligence

2. Click on the tool you want
   - Read the description
   - Check supported file formats

3. Upload your file(s)
   - Drag & drop to upload
   - Or click to browse files
   - Multiple files supported for Merge/Compare

4. Configure options (if available)
   - Some tools have settings
   - Adjust as needed

5. Process
   - Click "Convert", "Merge", etc.
   - Wait for completion
   - Usually instant (few seconds max)

6. Download
   - Click "Download" button
   - File saves automatically
   - Check your Downloads folder
```

### Examples

**🔗 Merge PDFs:**
1. Select "Merge PDF"
2. Upload 2+ PDF files (order matters!)
3. Click "Merge"
4. Download merged file

**🔐 Password Protect:**
1. Select "Protect PDF"
2. Upload PDF
3. Enter password (min 4 characters)
4. Click "Protect"
5. Download protected PDF

**📊 Convert PDF to Excel:**
1. Select "PDF to Excel"
2. Upload PDF (works best with tables)
3. Click "Convert"
4. Download .xlsx file
5. Open in Excel and edit

**📝 Add Watermark:**
1. Select "Watermark"
2. Upload PDF
3. Enter watermark text
4. Choose position, transparency, color
5. Click "Watermark"
6. Download marked PDF
